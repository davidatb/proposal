# proposal

Propuesta de graficos interactivos en web con js y la libreria chartJS

Generalidades

Este código contiene un arreglo de colores llamado "borders" y una función map que utiliza ese arreglo para crear un nuevo arreglo llamado "backgroundColors" con los mismos colores pero con transparencia. También contiene un objeto "data" que se utiliza para crear un gráfico y dos objetos que se utilizan como plugins para el gráfico. El primer objeto, "scrollButtons", añade dos botones de scroll a la gráfica que permiten desplazar el gráfico hacia la izquierda o derecha. El segundo objeto, "background_plugin", permite cambiar el color de fondo del gráfico.
